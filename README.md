# progetto-esame

## Compilare con Cmake
Dalla cartella principale del progetto (progetto-esame):
```bash
cmake -S . -B ../build
``` 
Verranno creati del file nella cartella build

Spostati nella cartella build e compila il progetto:
``` bash
cd build
make
```
